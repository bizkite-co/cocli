# Implement Event Curation TUI

Create a high-density TUI (using Textual) to review and curate events from the events WAL. 1. Display event details and hero images. 2. Support Approve, Reject, and Edit actions. 3. Approved events should be moved to an approved directory for newsletter ingestion. 4. Reject events that are out-of-area or not aligned with the Value-First policy (e.g. non-family friendly, high cost).
