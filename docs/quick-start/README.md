# Quick Start Guides

Welcome to the `cocli` Quick Start documentation. These guides are designed to help you perform common operational tasks quickly and efficiently.

## Operational Flows

- **[Main Scrape Flow](main-scrape-flow.md)**: How to set up, enqueue, and run a large-scale Google Maps scraping mission.

## Cluster Management

- **Check Health**: Run `python3 scripts/check_cluster_health.py` to see the status of all Raspberry Pi nodes.
- **Deploy Credentials**: Use `cocli infrastructure deploy-creds` to update AWS keys on the workers.
